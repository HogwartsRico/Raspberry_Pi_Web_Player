Raspberry_Pi_Web_Player
=======================

基于web.py开发树莓派播放器, 用网页进行控制

===
###需要
1.安装[webpy](http://webpy.org)  `sudo easy_install web.py`  
2.安装[mplayer](http://www.mplayerhq.hu)

###配置

修改main.py和player.py中的base_dir, 指向存放音乐的文件夹
